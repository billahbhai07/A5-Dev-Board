Question-1: What are the different ways to select an element in the DOM?

Ans: There are several ways to select elements in the DOM. The most common methods include:

  1. getElementById(): Selects by ID.
  2. getElementsByClassName(): Selects by class name.
  3. getElementsByTagName(): Selects by tag name.
  4. querySelector(): Selects the first element matching a CSS selector.
  5. querySelectorAll(): Selects all elements matching a CSS selector.
  6. childNodes: Selects all child nodes of an element.
  7. parentNode: Selects the parent node of an element.
  8. closest(): Selects the closest ancestor that matches a selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans : innerHTML, innerText, and textContent are all used to get or set the content of an element in the DOM,

  • innerHTML: Gets or sets HTML content, including tags.
  • innerText: Gets or sets visible text, considering styles and layout.
  • textContent: Gets or sets all text content, ignoring styles and layout.

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a technique where you attach a single event listener to a parent element instead of each child element. The event bubbles up, and the parent handles it, making it efficient and useful for dynamic content.

Question-4: What is event bubbling in the DOM?

Ans : Event bubbling is when an event starts at the target element and bubbles up to its parent elements, allowing parent elements to handle events without needing separate listeners on each child.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans : To create an element, use document.createElement(tagName). To add it to the DOM, use appendChild(). To remove an element, use removeChild() or remove().
